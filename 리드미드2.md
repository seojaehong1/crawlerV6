## 4대 PC 패턴 학습 실행 매뉴얼

### 공통 준비 (각 PC에서 한 번만 실행)
```powershell
# PowerShell 관리자 권한 권장
chcp 65001

# 프로젝트 경로로 이동 (필요에 따라 수정)
Set-Location "C:\Users\DU\Desktop\V5"

# 가상환경 생성 및 활성화
python -m venv venv
.\venv\Scripts\Activate.ps1

# 의존성 설치
pip install -r .\require.txt
playwright install

# 한글 로그가 깨지지 않도록 UTF-8 고정
$env:PYTHONUTF8 = "1"

# 결과 저장 폴더 생성
New-Item -ItemType Directory -Force -Path ".\patterns" | Out-Null
```

> 위 과정을 마치면 같은 터미널을 유지한 채로 해당 PC 블록을 실행하세요. 각 PC는 자신의 하위 폴더(`patterns\PC1` 등)에 결과 JSON이 쌓입니다.

### 실행 메모
- 오류 발생 시 `Ctrl + C` 후 5초 기다리고 같은 명령을 다시 실행하면 됩니다.
- 학습 결과 JSON은 `patterns\PC*\` 폴더 안에 쌓이며, 파일 이름의 앞 두 자리는 순번입니다.
- 필요 시 `--delay-ms` 값을 조절해 네트워크 상황에 맞게 대기 시간을 늘리거나 줄일 수 있습니다.
- PASS1만 수행하므로 결과 JSON은 체크마크 키 목록이 비어 있을 수 있습니다. PASS2 로직 작성 시 동일한 구조를 참고하세요.
